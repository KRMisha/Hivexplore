# Server

## Prerequisites

- [Python 3](https://www.python.org/downloads/)
- Install the required packages with a venv:

    ```
    python3 -m venv .venv
    source .venv/bin/activate # or .venv\Scripts\activate.bat on Windows
    pip install -r requirements.txt
    ```

## Usage

- `make run`
- `make test`
- `make lint`
- `make format`
