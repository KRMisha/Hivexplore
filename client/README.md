# Client

## Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)
- [Vue CLI](https://cli.vuejs.org/):

    ```
    npm install -g @vue/cli
    ```

## Project setup
```
npm install
```

npm run build
