# Server

## Prerequisites

- [Python 3](https://www.python.org/downloads/)
- Install the required packages with a venv: `make venv`

## Usage

Activate the venv before running any commands:
```sh
# Unix:
source .venv/bin/activate

# Windows:
.venv\Scripts\activate.bat on Windows
```

Commands:
- `make run`
- `make test`
- `make lint`
- `make format`
- `make help`
